중앙회노조 ON 웹앱 v7 - Google Forms/Sheets 실전형

1. 실행
- index.html을 브라우저에서 열면 됩니다.
- 서버 없이도 홍보용 화면은 바로 실행됩니다.

2. 의견수렴 실전 운영 방식
- v7은 Google Forms/Sheets 연동을 기본 방향으로 설계했습니다.
- 조합원이 의견 버튼을 누르면 Google Form으로 의견을 제출합니다.
- 제출된 의견은 Google Forms 응답 화면 또는 연결된 Google Sheets에서 관리자가 확인합니다.

3. 관리자 설정
- 관리자 메뉴 접속 비밀번호: 2026
- 관리자 > 의견 탭에서 다음을 수정할 수 있습니다.
  1) 의견 저장 방식: Google Forms/Sheets 사용 / 이 기기 저장
  2) Google Form 조합원 제출 링크
  3) Google Sheet 관리자 확인 링크
  4) 제출 버튼 문구
  5) 안내 문구
  6) 버튼형 표시 / 폼 삽입형 표시

4. 관리자가 바뀌는 경우
- 새 관리자는 관리자 모드에서 Google Form 링크와 Sheet 링크만 바꾸면 됩니다.
- 코딩 작업은 필요 없습니다.
- 모바일에서도 설정 가능합니다.

5. 권장 운영
- 실제 선거 운영: Google Forms/Sheets 사용
- 시연/테스트: 이 기기 저장 사용
